# 🎨 Module 5: Frontend Design System & Discovery Hub

**Assigned Member**: Ashish  
**Role**: UI/UX & Discovery Lead

---

## 📋 Responsibilities & Scope
1. **Glassmorphism Design System (`assets/css/style.css`)**: Glass tokens, frosted blur cards, rounded corners, soft glow shadows, dark mode stylesheet.
2. **Master JavaScript Engine (`assets/js/app.js`)**: GSAP entry animations, AOS initialization, button ripple effects, toast notifications.
3. **Shared Navigation & Layout (`includes/`)**: Floating glass header, brand logo, mobile hamburger offcanvas menu, glass footer.
4. **Discovery & Information Pages**:
   - `pages/home.php`: Hero banner, trending recipe carousel, popular categories, popular chefs showcase.
   - `pages/explore.php`: Live filter controls (search keyword, category, difficulty, sorting).
   - `pages/categories.php`: Categories grid with icons and recipe counts (`classes/Category.php`).
   - `pages/about.php`, `contact.php`, `privacy.php`, `terms.php`, `404.php`.
