# ⏱️ Module 4: Recipe Detail & Interactive Cooking Studio

**Assigned Member**: Jay  
**Role**: Cooking Studio & Interactions Lead

---

## 📋 Responsibilities & Scope
1. **Recipe Detail Page (`pages/detail.php`)**: Recipe view with hero backdrop, interactive ingredients checklist, nutrition specs, and instructions.
2. **Fullscreen Interactive Cooking Mode (`assets/js/cooking_mode.js`)**: Fullscreen overlay modal with step countdown timers, play/pause controls, progress bar, and audio finished alert.
3. **Interactions Engine (`classes/Interaction.php`)**: Like toggle, Bookmark toggle, Comments system with 5-star ratings.
4. **Reusable Recipe Card Component (`components/recipe_card.php`)**: Frosted blur glass card layout with hover lift.
