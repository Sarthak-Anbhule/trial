# 🍳 Module 2: Recipe Engine & Creator Studio

**Assigned Member**: Vedant  
**Role**: Recipe CRUD & Creator Studio Lead

---

## 📋 Responsibilities & Scope
1. **Recipe Engine (`classes/Recipe.php`)**: Recipe creation, deletion, editing, database queries, category filter queries.
2. **Recipe Upload Studio (`pages/upload.php`)**: Form interface for creating new recipes with cover image upload.
3. **Dynamic Form Builder (`assets/js/recipe_builder.js`)**: Dynamic addition and removal of ingredient rows and cooking step rows.
4. **Live Search API (`api/search.php`)**: Rapid JSON endpoint for live search suggestions.
