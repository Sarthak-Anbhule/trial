# 👤 Module 3: User Social & Profile Engine

**Assigned Member**: Aryan  
**Role**: Social Features & User Profile Lead

---

## 📋 Responsibilities & Scope
1. **User Profile Engine (`classes/User.php`)**: User profile fetching, avatar/cover photo upload handlers, follower/following statistics.
2. **Follow/Unfollow API (`api/follow.php`)**: Creator follow toggle.
3. **User Pages**:
   - `pages/profile.php`: Public chef profile & portfolio view.
   - `pages/edit_profile.php`: Edit avatar, bio, and display name.
   - `pages/saved.php`: Personal saved bookmarks gallery.
   - `pages/my_recipes.php`: Personal uploaded recipes manager.
   - `pages/notifications.php`: User activity stream (`classes/Notification.php`).
